Installation:

  1. Run pip install djwebsockets.
  2. Add djwebsockets to settings.INSTALLED_APPS
  3. Add WEBSOCKET_HOST and WEBSOCKET_PORT to settings.py
  4. In wsgi.py file, replace line
     from django.core.wsgi import get_wsgi_application
    with
     from djwebsockets.wsgi import get_wsgi_application
  5. optional: Use following for session variables and user (websocket.user and websocket.session)
     WEBSOCKET_MIDDLEWARE = [
     'django.contrib.sessions.middleware.SessionMiddleware',
     'django.contrib.auth.middleware.AuthenticationMiddleware',
 ]
